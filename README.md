# Warrantia
Ironhack - Project 2 - CRUD web app

Warrantia is an app that manages your warranties for all your products.  


## Features
- Create, read, upload & delete a product and its associated warranties.
- Track all your products purchases, associated warranties and their expiry dates.
- Features different warranty types (builder, seller (extension), payment provider, insurance).
- allows you to maintain an inventory of all your purchases.

## Next steps

- Improve global user experience and readability.
- Improve login/sign-up experience (security, password management).
